# Insurance Management System

This repository contains a front-end insurance management project with HTML, CSS, and JavaScript.

##video clip -
- https://youtu.be/00qwtKWsQkw

## Overview

- `css/`: Stylesheets for admin, customer, insurance manager, and other pages.
- `js/`: JavaScript files used for validation, page interactions, printing, and rating.
- `images/`: Project images and assets.
- `Website/`: Public-facing website pages and assets.

## Key Features

- Admin dashboard and management pages
- Customer registration, login, and profile handling
- Insurance manager pages for reporting, claims, and payments
- Web pages for insurance plans, offers, contact, and home page

## How to Run

1. Open the project folder in your browser.
2. Use one of the HTML pages from the `Website/` folder, for example:
   - `Website/Home page.html`
   - `Website/LoginPages.html`
   - `Website/PlanMain.html`
3. If you need to test more UI pages, open the relevant HTML file from the project folder.

## Notes

- This project appears to be a static site with client-side scripts.
- No backend or package manager setup is required unless you add one later.

## Publishing to GitHub

1. Initialize Git in the folder if not already initialized:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. Create a repository on GitHub.
3. Add the remote and push:
   ```bash
   git branch -M main
   git remote add origin https://github.com/<your-username>/<your-repo>.git
   git push -u origin main
   ```
